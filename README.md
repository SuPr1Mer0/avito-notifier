# Telegram Avito Notifier с базой данных

Простой, но демонстрационный Telegram-бот на Python, который:
- Сохраняет вставленную ссылку в базу данных
- Проверяет наличие новых объявлений
- Уведомляет о новых объявлениях

## Возможности
- FSM-диалог при добавлении подписки: название → URL/запрос → цена мин → цена макс → ключевые слова
- Фильтрация по цене и ключевым словам прямо в боте
- SQLite без внешних серверов БД
- Лимит 5 уведомлений за цикл на подписку (защита от флуда)
- Отключение уведомлений кнопкой в меню
- Автоматическая очистка старых записей (30 дней)

## Стек
- pyTelegramBotAPI
- SQLite3
- python-dotenv
